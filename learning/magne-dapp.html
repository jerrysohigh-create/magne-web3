Magne dApp

The Magne dApp is the gateway to the MAGNE ecosystem, providing users with an intuitive interface to manage their digital assets, access DePin services, and interact with decentralized applications.

Key Features

Wallet Management

  • Create and manage MAGNE wallet addresses
  • Securely store $MHA and other tokens
  • View transaction history and balances
  • Multi-signature support for enhanced security

DePin Services

  • Share idle device compute for network rewards
  • Monitor contribution statistics
  • Manage device registrations
  • Access AI services powered by the network

Token Swap

  • Decentralized exchange (DEX) integration
  • Swap tokens with low fees (0.05%-1%)
  • Provide liquidity and earn yields
  • Access liquidity pools

Staking & Delegation

  • Stake $MHA to earn rewards
  • Delegate to validators
  • Track staking performance
  • View expected APY

Accessing the dApp

The Magne dApp can be accessed through:

  • Web: Visit dapp.magne.ai (when available)
  • Mobile: MAGNE Phone Gen1 built-in wallet
  • Browser Extension: Coming soon
Security Tip: Never share your private keys or seed phrases. The MAGNE team will never ask for your credentials.
