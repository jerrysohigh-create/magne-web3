如何連接至 MAGNE 測試網

為開發者提供的指南，說明如何將 MAGNE L1 及 M Hash L2 測試網新增至 MetaMask 或其他 Web3 錢包。

1 開始之前

⚠️ 僅限測試網

以下網路僅為測試網。參數可能會有變動。請勿在測試網上使用真實資產或正式錢包。測試網代幣不具貨幣價值。

  • RPC 端點、鏈 ID 及區塊瀏覽器 URL 可能會更新
  • 請使用專用測試錢包進行開發
  • 交易前請務必驗證網路詳情
  • 如有需要，請聯繫 @MagneAI
2 將 MAGNE L1 測試網新增至 MetaMask

在 MetaMask 中，進入設定 → 網路 → 新增網路，並輸入以下資訊：

網路名稱 MAGNE L1 Testnet
RPC URL https://rpc.testnet.magicalhash.com
鏈 ID 20250810
代幣符號 MHA
區塊瀏覽器 https://explorer.testnet.magicalhash.com
查看 L1 瀏覽器
3 將 M Hash L2 測試網新增至 MetaMask

在 MetaMask 中，進入設定 → 網路 → 新增網路，並輸入以下資訊：

網路名稱 M Hash L2 Testnet
RPC URL https://l2-rpc.testnet.magicalhash.com
鏈 ID 20250827
代幣符號 MHA
區塊瀏覽器 https://l2-explorer.testnet.magicalhash.com
查看 L2 瀏覽器
4 取得測試網 MHA

連接完成後，您可以取得測試網 MHA 代幣以進行開發及測試。

水龍頭開發中

測試網水龍頭目前仍在開發中。如需測試網 MHA，請聯繫 @MagneAI。

5 驗證連線
  • 檢查網路：確認您的錢包顯示正確的網路名稱及鏈 ID
  • 查看地址：前往區塊瀏覽器並搜尋您的錢包地址
  • 發送測試交易：嘗試向自己或其他測試地址發送少量代幣
  • 確認收款：驗證交易是否出現在瀏覽器中

常見問題

RPC 無回應

請檢查您的網路連線。若 RPC 端點正在維護中，請稍後再試。您也可以透過官方頻道查看網路狀態。

鏈 ID 不匹配錯誤

請確認您輸入的鏈 ID 正確：L1 為 20250810，L2 為 20250827。部分錢包要求輸入數值型鏈 ID，且不可包含前導零。

水龍頭無法運作

水龍頭仍在開發中。如需測試網代幣，請聯繫 @MagneAI。

交易待處理

在高流量期間，測試網交易可能需要更長時間。請在瀏覽器中查看待處理狀態。L2 交易通常比 L1 更快確認。

選錯網路

在發送交易前，請確認您的錢包已切換至正確的網路（MAGNE L1 或 M Hash L2）。

⚠️ 安全注意事項

測試網代幣不具貨幣價值。請勿將正式錢包或私鑰不必要地導入測試網。請使用專用測試錢包進行開發。

MAGNE.AI 團隊絕不會要求您提供私鑰或助記詞。請妥善保管您的憑證，切勿與任何人分享。
