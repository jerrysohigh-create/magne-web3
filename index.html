MAGNE Agent Pay is a verified testnet-stage demonstration featuring M Hash L2 contract deployment and AI Task Receipt recording. Full HTTP 402 payment flow, browser wallet authorization, and Facilitator verify/settle remain in development. Formal x402 compatibility remains under validation.

Hardware Authorization

Turn secure mobile hardware into an authorization endpoint for agent payments.

MAGNE.AI Phone integrates a Secure Element (eSE) and TEE runtime. Authorization requests are signed cryptographically — without exposing private keys.

🔐

Secure Element (eSE)

Hardware-level key storage within the secure chip. Signing keys are designed to remain within secure hardware. Authorization is cryptographically enforced.

📱

TEE Runtime

Trusted Execution Environment. Authorization logic is intended to run within an isolated execution environment, designed to reduce exposure to the main operating system and applications.

🔑

x402 Compatible

Designed for future interoperability with x402-compatible APIs, subject to implementation and compatibility testing.

👤

Human Override

Designed to allow the account holder to review and approve each authorization request. Authorization workflows are intended to be user-controlled.
