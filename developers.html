Code Examples

ethers.js Connection

const { ethers } = require('ethers');

const provider = new ethers.JsonRpcProvider(
  'https://rpc.testnet.magicalhash.com'
);

const wallet = new ethers.Wallet(
  process.env.TESTNET_PRIVATE_KEY,
  provider
);

const balance = await provider.getBalance(wallet.address);
console.log('Balance:', ethers.formatEther(balance), 'MHA');

// Security: Use a dedicated test wallet.
// Never expose private keys in frontend code or public repositories.

Agent Pay SDK

x402-Compatible Agent Payment

TESTNET-STAGE

MAGNE Agent Pay is a developer demo for AI agent payment on M Hash L2. Agents call a service → service returns HTTP 402 with payment metadata → wallet authorizes mMHA transfer → facilitator verifies on M Hash L2 → AI Task Receipt created on-chain.

The flow above is an integration reference architecture. Publicly disclosed testnet evidence currently covers identified contract deployments and AI Task Receipt recording. HTTP 402 payment handling, browser wallet authorization and Facilitator verify/settle logic remain in development or under validation.

Payment Flow Code

// 1. Agent calls service API
const response = await fetch('https://api.magne.ai/wallet-risk', {
  method: 'POST',
  body: JSON.stringify({ wallet: '0x742d...' })
});
// 2. Service returns 402 with payment metadata
// { requiredAmount: '0.01', token: 'mMHA', chain: 'mhash-l2' }

// 3. Wallet authorizes mMHA transfer
const signer = await provider.getSigner();
const tx = await mMHA.connect(signer).transfer(recipient, amount);
await tx.wait();

// 4. Facilitator verifies payment strictly
const result = await facilitator.strictVerify({
  paymentTxHash: tx.hash,
  taskId: 'task_001',
  expectedAmount: parseEther('0.01'),
  expectedRecipient: recipient
});
// Returns { status: 'verified' } or { status: 'failed', reason: '...' }

// 5. AI Task Receipt created on M Hash L2
// { receiptId, taskId, amount, timestamp, paymentTxHash }
