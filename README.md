## Agent Pay 定位

MAGNE Agent Pay 是面向 AI 代理支付場景的開發階段演示，結合 M Hash L2 測試網上的合約部署與 AI 任務收據記錄能力。

正式 HTTP 402 支付流程、瀏覽器錢包授權、Facilitator 驗證與結算流程，以及 x402 相容性測試仍在開發與驗證中。

Agent Pay 是旗艦開發者演示頁，不是整個 Web3 入口的首頁。首頁是以 Agent Pay 為核心敘事的主頁門戶。
